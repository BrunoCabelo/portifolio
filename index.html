Projetos

Data4Crops

Este foi um proejeto realizado para a startup Data4Crops, foi desenvolvido um site landing-page completo com a utilização de HTML5, CSS3, JavaScript e Bootstrap. Além do desenvolvimento de um sistema de login completo que conta com confirmações de cadastro e alteração de senha via e-mail, e criptografia em banco de dados.

Todo o desenvolvimento de back-end foi feito a partir do zero utilizando PHP7; toda a modelagem do banco de dados foi feita via MySQL.

Link: https://www.data4crops.com.br/


xPaper

Esse foi o meu primeiro projeto, foi desenvolvido como sistema interno da Eucatex, e seria utilizado para monitorar o consumo de bobinas de papel em diferentes maquinas da fábrica; esse projeto conta a manipulação de dados JSON e utilização de DataTables.

Todo o desenvolvimento back-end foi feito utilizado PHP7; a modelagem do banco de dados relacional foi feita utilizado MySQL e o MariaDB.

Link: https://github.com/BrunoCabelo/xPaper


Editor de texto

Esse projeto foi desenvolvido somente para desenvolver minhas hábilidades de CSS3 e JavaScript. 

Todas as funcionalidades foram desenvolvidas com a utilização de JQuery, além disso todo o design foi feito com CSS3 puro; tudo isso mantendo as boas práticas de organização do código.

Link: https://editor-de-texto-six.vercel.app/


Kanban funcional

O projeto do Kanban foi feito durante um curso de JavaScript avançado e teve como principal objetivo conhecer métodos avançados de desenvolvimento com JavaScript.

Esse projeto conta com a criação de uma API JavaScript e a utilização do modelo MVC; ele também conta com a utilização das funções de armazenamento em localStorage. Todo o design foi feito com a utilização de CSS3 puro.

Link: https://github.com/BrunoCabelo/Kanban-board
